### 1.1 算法

> ::: info 算法
>
> <b>算法</b>就是字面意思，就是我们常说的数学解题方法。
>
> 比如：`1+2+3+...+99+100=?`
>
> - 算法1（解法1）：
>   - `1+2=3，3+3=6，6+4=10，...，加到100 ---> 5050`
> - 算法2（解法2）：
>   - `（1+100）* 50 / 2= 101 * 50 = 5050`   即：高斯算法
>
> <b>算法</b>评价的具体的指标：（了解，后续再详解）
>
> - 时间复杂度
> - 空间复杂度
>
> :::
